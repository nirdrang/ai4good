#### REQ-005.5: Project Lifecycle State-Transition Table

Nine states: `draft`, `discovery_in_progress`, `scoped`, `triage`, `open`, `matched_pending_fuel`, `in_progress`, `completed`, `cancelled`. Every transition has an actor, preconditions, side effects, and failure handling. Abandonment/rematch (`in_progress â†’ open`, REQ-027) is a transition, not a tenth state.

> **No `paused` state in v1.** A pause is a support conversation: pre-match, the NGO unpublishes or cancels; mid-build, an admin revokes access with a note, or the project is cancelled. (â†’ RM-17)
